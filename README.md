# QuestAnnouncer
Automatically shows the progress/status of quests in the party chat.

## Description
A Turtle WoW addon that announces quest updates in the party chat to make communication easier.

## Usage
- `/qa test` → (must be in a party) sends a fake quest update to verify that the addon works.

## Example
![Example](/Images/QuestAnnouncer.png)

## Installation
1. Open Code dropdown menu (slightly to the top right)
2. Select Download as ZIP
3. Extract downloaded zip into WoW/Interface/AddOns
4. Rename folder to Gatherer from Gatherer-master
