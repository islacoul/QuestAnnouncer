# QuestAnnouncer
Automatically shows the progress/status of quests in the party chat.

## Description
A Turtle WoW addon that announces quest updates in the party chat to make communication easier.

## Usage
- `/qa test` → (must be in a party) sends a fake quest update to verify that the addon works.

## Installation

1. Download the latest `.zip` from the \[Releases](https://github.com/YourUsername/QuestAnnouncer/releases).  
2. Extract it into your WoW addon directory:  
	- `World of Warcraft/Interface/AddOns/QuestAnnouncer/`  
3. Enable the addon in the AddOns menu in-game.
